# 캡스톤디자인1 프로젝트
## 프로젝트 개요
**프로젝트 명**: 아프지마요

**프로젝트 배경**: 
> 코로나로 병원 방문 12% 감소…진료비는 되레 늘어
> 23일 뉴시스가 보건복지부와 건강보험심사평가원의 '진료비 주요통계'를 분석한 결과 지난해 상반기 내원일수는 6억8260만 일로 전년 동기 7억7983만 일에 비해 12.5% 줄었다. 분기별로는 코로나가 확산됐던 2분기에 17.4%나 줄어드는 등 감소세가 두드러 졌다. 1분기는 7.1% 감소에 그쳤다. 

코로나의 장기화로 인하여 병원을 기피하게 되어 자신의 병의 증세를 자가 판단을 통하여 병원을 방문할지를 결정하는 경우가 많다.
하지만 이러한 자가 판단은 정확하지 않고, 실제로 건강이 이상이 있어도 병원을 방문하지 않는 경우가 있다.
따라서 정말로 건강에 이상이 있다고 생각하는 사용자가 병원에 안 가는 문제점을 해결하고자 했다.

**프로젝트 내용**: 
사용자가 자신의 증상에 대해서, 크게 장기별로 나누어서 선택할 수 있도로 하고, 
간단한 객관식 설문지를 통하여 해당 장기별 증상에 대하여 답변한다. 
이렇게 작성한 답변지를 증상-병명의 데이터를 통해 머신러닝을 통하여 가장 근접한 병명에 대한 답을 제공하여, 사용자가 병원을 방문할 필요가 있는지에 대한 가이드르 제공한다.
추가적으로 병원을 갈 필요가 있는 경우, 위치 데이터를 확인하여 근처의 해당 병의 진료과와 병의 크기에 따라서 적절한 병원을 알려준다.
